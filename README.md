# IMDB Movie Lister

## Setup
Installing deps
```bash
bun install
```

Starting dev Server
```bash
bun run dev
```

Your localinstance is up click following link
http://localhost:5173


## TechStack Used
- React
- Typescript
- Tailwindcss
- Shadcn
- Vite
- Firebase
